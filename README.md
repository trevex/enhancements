# Enhancements

This is the home of the IronCore enhancement / extension proposals. You can find
the list of accepted proposals in the [ieps/ directory](ieps/).

## Submitting a New Enhancement Proposal

Please use our [template](ieps/00-template.md) to write and submit your proposal
via a pull request.

> [!Note]
> After approval and before merging, the next consecutive number must be assigned to the IEP.
